# ⚡ Energy Consumption and Cost Calculator

## 📒 About this project
This project was created to provide an estimate for energy consumption and cost.

## ❓ How to use it?
The user needs to inform an **appliance**, its **power** in watts (W), **daily usage** in hours and the **cost** per kWh.

## ✨ How does it work?
The program multiplies the power by the daily usage and the amount of days in a month (rounded up to 30 in this case) to calculate the average energy consumption.

Furthermore, we multiply the average energy consumption by the cost per kWh to find the final cost per month.
